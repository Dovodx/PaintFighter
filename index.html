Paint Fighter: The Video Game

Fight to your badly-drawn death with amazing characters like Dr. Tentacle, Sad Sword, Liquid, and Sanic teh Hegehog!
The game is currently in development with more characters and levels planned!
This website is also a work in progress.

Current State

This is a multiplayer fighting game that is, essentially, an "mspaint" parody of Super Smash Bros. The game supports up to four players both locally and online. You can even have more than one player on the same computer when playing online!

Here are the characters currently in the game:

Dr. Tentacle
Sad Sword
Liquid
Sanic
Rock Lobster
Thundersnow
fappy tails
Shaydoh
Beowulf

As for levels:

  • testroom (will be renamed and restyled later)
  • Snowy Summit
  • Sanic Land (troll level)
  • Desktop 2000
  • RGB Land
  • The Aeroplane
  • Westopolis
  • Party Bus
The current version of the game is Alpha v11.

In the future, I hope to host a download link for the latest version on this site. The game's "Alpha" state means that there are still bugs being found and there is lots of planned content that is not yet in the game. Alpha testing is just among my friends and I for now.

  • 9-4-18

    WOW I suck at updates lol. I've added a tiny bit of info on the new update. There will be two new levels in addition to a bunch of other things I've been working on. I mentioned a new menu theme last update, but that's being put to the wayside for now (I hadn't made much of the song, anyway). I've been messing with game balance things like gravity and hitstun a lot. I've also made an entirely new level select screen. It's barebones, but it's more usable than the two arrow button system the game used before. I've added a bunch of small things, like a visual effect for invincibility, the number of lives can now be changed, and levels can specifically have kill boundaries at the top that kill you regardless of whether you've been hit. That last one only really affects Sanic Land for now (the ten mile long troll level) because it has spikes at the top. I moved character controls to its own script so I don't have to copy paste the control code for every character. This means that making new characters will be negligably faster since I don't have to paste and edit spaghetti messes of code for each one anymore. More importantly, it will allow me to make AI players that control their characters the same way humans do, effectively. I plan for AI players to be in a future version, like 12 or 13 hopefully. I've also fixed a bunch of bugs, like Sad Sword getting stuck on the back of the Aeroplane when doing his Air Normal kick attack. He also can't camp inside the colliders of levels anymore (though Snowy Summit still needs a little more tweaking). There were also bad problems with the camera being stupid, grab meters not showing or behaving properly, and the game just ending if someone didn't load fast enough. There will be more detail on these fixes and changes when I finally finish this damn update lol. All I can do right now is assure my friends (the only people reading this site other than me) that there is a SECRET sauce in this update that I'm staying tight-lipped about. You shall see...
